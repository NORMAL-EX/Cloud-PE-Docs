# 插件投稿
:::tip 注意
<strong>上传时请把插件命名成以下格式：（使用 [Cloud-PE插件助手](https://p0.cloud-pe.cn/Cloud-PE%E6%96%87%E4%BB%B6/CE-%E6%8F%92%E4%BB%B6%E5%88%B6%E4%BD%9C%E5%B7%A5%E5%85%B7/CEMake.exe) 生成出的插件默认即是该格式）</strong>
```
[软件分类]软件名_作者（你）_软件版本号_插件介绍.ce

如：
[安全维护]维护工具_1.0.1_dddffgg_Cloud-PE官方推出的PE维护工具包.ce
```
请确保插件:未重复（可为新版）、路径使用相对路径或环境变量、无害，文件名不超过80个字符，可在 Cloud-PE 内正常加载


上传插件后，会对你的插件进行评估，通过则加入插件市场
:::

:::warning 可选软件分类如下：

运行环境、远程控制、游戏娱乐、硬件检测、系统工具、文件工具、网络浏览、输入法、视频音乐、实用工具、社交通讯、上传下载、驱动程序、其他软件、教育学习、磁盘工具、编程开发、备份还原、办公编辑、安全维护
:::

:::danger 警告
含有压缩包密码的插件将不予审核通过，上传前请自行确认你的插件无压缩包密码
:::
